# LinkedIn Queens Puzzle Solver

Solving the puzzle game Queens on LinkedIn. Currently in initial stages.

## Setup

1. Create and activate a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Install Playwright browsers (required to download levels):
   ```bash
   playwright install
   ```
